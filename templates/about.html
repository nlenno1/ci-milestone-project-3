About

How to use and get the most out of MovieWiki!

Welcome to MovieWiki, the place to share your views on the latest movies and find information on new movies to watch!

To get started have a look round to see what Movies and Reviews are already here and if you want to add to MovieWiki, just create an account or sign in!
Below is our FAQs if you need any help, but if something isn't covered then send us a message throught the Contact button above!

FAQs

You have to be over the age of 8 to create an account on MovieWiki however anyone can view the Movies and Reviews

With a MovieWiki Account, you can:

  • Add Movies and update the Movies You create
  • Add Reviews to existing Movies
  • Keep track of Movies you have reviewed in your User Profile
  • Create a list of Movies you have watched and Movies you want to watch which is displayed in your User Profile and on the Movies

What are you waiting for? Sign Up!

You can add any Movie you like as long as it has been released already. You cannot add Movies that are yet to be seen publically.

To add a Movie:

  1. Click Movies at the top of the page
  2. Select "New Movie"
  3. Fill out all required fields (*) in the form
  4. Click Create Movie

You need to be signed in to create a Movie

Adding a Movie Image Link isn't required, when creating a movie, however if you want to add one, this is how you do it:

  1. Find a movie image online that you like
  2. Check that it is royalty free and using it will not infringe copyright laws
  3. Right click on the image and select "copy image address". If there is not this option then left click on the image and try again
  4. Paste the link into a url bar to check that it works
  5. If there is any text after the image extension (eg. .jpg or.png) then remove it in the URL bar and check the link again. If the link breaks then find a new image
  6. Copy the image link into the input box

You will find the Update Movie button under the Movie name in the Movie's profile, however this button will only appear if you created the Movie.
If you see some incorrect information in a Movie Profile then get in touch with us using the Contact button above.

The Delete Movie button can be found at the bottom of the Update Movie form

You will find your Watched and Want to Watch lists displayed in your User Profile

The buttons to control these list are underneath the Movie Title in their individual Movie Profiles. The Icon next to the Movie Name, will tell you if you have added the current Movie to either list

To add a Review:

  1. Find a Movie you want to Review and click on its profile
  2. Click the "Create Review" button
  3. Fill out all required fields (*) in the form
  4. Click Create Review

You need to be signed in to create a Review

To Update your review, visit the Movie's profile and click the Update Review button at the top of the page

To Delete your review, click the View All Reviews button on the Movie's Profile and when the Reviews are listed, yours will have an Update and Delete button underneath the Review

To Update your account details, navigate to your profile page, click the Update Profile button underneath your Username and make any changes in the form before clicking Update Profile.

If you wish to Delete you account on Movie Wiki, you will find the button at the bottom of your Update Account page.

Deleting your account will be permenent and you will not be able to modify any reviews or movie profiles created with this account even if you register again
