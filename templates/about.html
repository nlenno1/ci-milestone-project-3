With a MovieWiki Account, you can:

  • Add Movies and update the Movies You create
  • Add Reviews to existing Movies
  • Keep track of Movies you have reviewed in your User Profile
  • Create a list of Movies you have watched and Movies you want to watch which is displayed in you User Profile and on the Movies

What are you waiting for? Sign Up!

To add a Movie:

  1. Click Movies at the top of the page
  2. Select "New Movie"
  3. Fill out all required fields (*) in the form
  4. Click Create Movie

You need to be signed in to create a movie

To add a Movie Image link:

  1. Find a movie image online that you like
  2. Check that it is royalty free and using it will not infringe copyright laws
  3. Right click on the image and select "copy image address". If there is not this option then left click on the image and try again
  4. Paste the link into a url bar to check that it works
  5. If there is any text after the image extension (eg. .jpg or.png) then remove it in the URL bar and check the link again. If the link breaks then find a new image
  6. Copy the image link into the input box
